# 前后端分离Demo

目录说明：

- backend：Go后端（Gin）

- frontend：前端（Vue）

- sql：数据表（SQLite）

### 1 运行环境

#### 1.1 Vue【前端】

##### 运行方式：

```shell
npm install

npm run serve
```

#### 1.2 Go【后端】

##### 运行方式：

```shell
go mod tidy

go run main.go
```
